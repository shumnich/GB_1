# Rules of cmd

Команда смены директории

```sh
cd "path to folder"
```

Команда отображения текущей директории для MacOS, Linux
```sh
pwd 
```
Листинг текущей директории Windows:
```sh
dir
```
MacOs, Linux:
```sh
 ls
```

Команда удаления файла в Windows:
```sh
del "filename"
```
Команда удаления файла в MacOS, Linux:
```sh
rm "filename"
```
